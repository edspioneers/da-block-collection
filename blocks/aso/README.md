# ASO Block

The ASO (App Store Optimization) block provides a structured way to manage and display App Store Optimization metadata.

## Structure

The ASO block expects 4 rows with 2 columns each:

| Row | Column 1 | Column 2 |
|-----|----------|----------|
| 1 | Title | App/Product Title |
| 2 | Description | Detailed Description |
| 3 | Description Character Limit | Character limit info |
| 4 | Developer | Developer name or company |

## Fields

- **Title** (required): The title of your app or product
- **Description** (required): A detailed description of your app
- **Description Character Limit**: Information about description character limits
- **Developer**: The name of the developer or company

## Usage

Simply insert the ASO block into a section and fill in the required metadata fields through the editing interface.
